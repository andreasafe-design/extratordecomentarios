Extrator de comentários

Exporte comentários de qualquer arquivo do Figma ou FigJam para Excel

Gere em figma.com/settings → Security → Personal access tokens.

Cole a URL completa do arquivo do Figma ou FigJam.

Filtre os comentários pelo período de criação. Se nenhum período for informado, todos os comentários do arquivo serão considerados.
